# ⚙️ Jenkins Practice Series

This repository contains a structured, day-by-day learning path for mastering **Jenkins**, one of the most popular tools for **CI/CD (Continuous Integration and Continuous Deployment)**.

Each folder represents a practical task designed to build real-world DevOps skills using Jenkins.

---

## 📚 Contents

### 🔹 Jenkins Fundamentals

* **Day 01**: Set Up Jenkins Server
* **Day 02**: Install Jenkins Plugins
* **Day 03**: Configure Jenkins User Access
* **Day 04**: Configure Job for Package Installation
* **Day 05**: Parameterized Builds
* **Day 06**: Database Backup Job

### 🔹 Intermediate Concepts

* **Day 07**: Jenkins Slave Nodes (Distributed Builds)
* **Day 08**: Project Security Configuration
* **Day 09**: Deployment Pipeline (Web App)
* **Day 10**: Conditional Pipeline (Jenkinsfile)
* **Day 11**: Deployment Job

### 🔹 Advanced Pipelines

* **Day 12**: Chained Builds
* **Day 13**: Multistage Pipeline

## ▶️ Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/abhijitray7810/Jenkins-Pratices-100.git
   ```

2. Navigate to a specific day:

   ```bash
   cd Day-XX
   ```

3. Follow the README instructions in each folder to execute tasks.
